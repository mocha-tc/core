# mocha

this repository hosts mocha's core packages. you may be looking for [mocha-tc/plugin](https://github.com/mocha-tc/plugin), which hosts mocha's roblox plugin.

## mocha can:
- convert tc client objects to v5.5 client objects
- convert a tc tower to an mtk v4 tower
- remember your settings for quick conversion

please see `CONTRIBUTING.md` for tips on how to contribute to mocha!
